Math Vocabulary

Key algebra terms to boost your understanding

Expressions & Equations

Variable

A symbol, usually a letter, that represents a number in mathematical expressions or equations.

Coefficient

The number that multiplies a variable in an algebraic expression.

Term

A part of an expression or equation separated by addition or subtraction.

Expression

A combination of numbers, variables, and operations that represents a value.

Equation

A mathematical statement that shows two expressions are equal.

Constant

A fixed value that does not change.

Distributive Property

a(b + c) = ab + ac; used to multiply a number by a sum or difference.

Inverse Operation

Operations that undo each other, such as addition and subtraction.

Linear Equation

An equation that graphs as a straight line.

Quadratic Equation

An equation of the form ax¬≤ + bx + c = 0.

System of Equations

A set of two or more equations with the same variables.

Substitution

Replacing a variable with a value or expression.

Elimination

A method for solving systems by adding or subtracting equations.

Standard Form

The usual way to write equations, such as Ax + By = C.

Simplify

To write an expression in its most reduced form.

Evaluate

To find the value of an expression.

Parentheses

Used to group parts of expressions or equations.

Functions & Graphing

Function

A relation where each input has exactly one output.

Domain

All possible input values of a function.

Range

All possible output values of a function.

Slope

The measure of steepness of a line, often represented as rise over run (m).

Y-Intercept

The point where a graph crosses the y-axis.

X-Intercept

The point where a graph crosses the x-axis.

Graph

A visual representation of data or mathematical relationships.

Number Properties

Absolute Value

The distance a number is from zero on the number line.

Rational Number

A number that can be expressed as a ratio or fraction.

Irrational Number

A number that cannot be expressed as a fraction.

Real Numbers

The set of all rational and irrational numbers.

Integer

All whole numbers and their opposites.

Natural Numbers

Counting numbers starting from 1.

Whole Numbers

Natural numbers including zero.

Polynomials & Exponents

Monomial

An algebraic expression with one term.

Binomial

An algebraic expression with two terms.

Trinomial

An algebraic expression with three terms.

Polynomial

An expression made up of terms that include variables raised to whole-number exponents.

Exponent

The number that indicates how many times a base is multiplied by itself.

Base

The number that is multiplied by itself when using exponents.

Exponent Rules

Rules used to simplify expressions involving powers.

Factors & Multiples

Factor

Numbers or expressions that multiply together to form a product.

Greatest Common Factor (GCF)

The largest factor shared by two or more numbers or terms.

Least Common Multiple (LCM)

The smallest multiple that two or more numbers have in common.

Factorization

The process of breaking down numbers or expressions into products of factors.

Prime Number

A number with exactly two distinct positive divisors: 1 and itself.

Composite Number

A number with more than two factors.

Radicals & Order

Radical

An expression that uses a root symbol, such as ‚àö.

Square Root

A value that, when multiplied by itself, gives the original number.

Order of Operations

The correct sequence to evaluate an expression: PEMDAS.

Inequality

A mathematical sentence showing that two values are not equal using symbols like <, >, ‚â§, ‚â•.

Solution

The value(s) that make an equation or inequality true.
